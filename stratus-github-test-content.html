🎯 Stratus Add-on GitHub Integration Test

This page tests the Stratus Mermaid add-on's GitHub integration with our FKmermaid repository.

📋 Setup Instructions:

  1. Install the Stratus Add-ons for Confluence from the Atlassian Marketplace
  2. In the Stratus add-on configuration, connect your GitHub account via OAuth
  3. Use the GitHub Repository Explorer (GRE) to navigate to: wellmindhealth/FKmermaid/test-diagrams/
  4. Select any .mmd file to render it in-page

🔗 Available Test Diagrams:

  • simple-flow.mmd - Basic flowchart diagram
  • er-diagram.mmd - Entity Relationship diagram
  • sequence-diagram.mmd - Sequence diagram

📁 GitHub Repository:

Repository: https://github.com/wellmindhealth/FKmermaid

Path: test-diagrams/

🎨 Expected Result:

When you select a .mmd file through the Stratus GitHub integration, it should:

  • ✅ Fetch the file content from GitHub
  • ✅ Parse the Mermaid syntax
  • ✅ Render the diagram in-page
  • ✅ Provide zoom/pan controls
  • ✅ Allow full-screen viewing

🔧 Troubleshooting:

If the integration doesn't work:

  1. Check that GitHub OAuth is properly connected
  2. Verify the repository is accessible (public or you have access)
  3. Ensure the .mmd files contain valid Mermaid syntax
  4. Try refreshing the page after selecting a file
